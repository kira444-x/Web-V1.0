lists

In HTML, lists are used to group related items together. There are three types of lists:

1. Ordered List ol

An ordered list is used to create a list of items that are numbered sequentially. Each item in the list is created using the li (list item) tag.

    exmple :
  1. Apple
  2. Banana
  3. Cherry

Attributes of ol

type: Specifies the type of numbering. Values include:
    1: Default, numbers (1, 2, 3, …)
    A: Uppercase letters (A, B, C, …)
    a: Lowercase letters (a, b, c, …)
    I: Uppercase Roman numerals (I, II, III, …)
    i: Lowercase Roman numerals (i, ii, iii, …)

start: You can specify starting number of the list using the start attribute:
ol start='5' (This will start the list from 5: )

2. Unordered List ul

An unordered list is used for items where the order doesn't matter. The items are marked with disc (default).

    exmple:
  • red
  • white
  • pink

Attributes of ul

type: Specifies the bullet style. Values include:
    disc: Default, filled circle
    circle: Unfilled circle
    square: Square bullet

3. Definition list dl

A definition list is used for terms and their descriptions. The terms are defined using the dt tag and the descriptions are provided with the dd tag.

exmple:
Term 1
Description of term 1
Term 2
Description of term 2
exmple:
HTML
A markup language for creating web pages.
CSS
A stylesheet language for styling web pages.

4. Nesting Lists

Lists can be nested inside other lists.

    exmple
  • Fruits
    • Apple
    • Banana
  • Vegetables
    1. Carrot
    2. Broccoli

Conclusion

        Use ol for ordered (numbered) lists.
        Use ul for unordered (bulleted) lists.
        Use dl for definition lists (terms and descriptions).
